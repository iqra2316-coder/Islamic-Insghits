# Islamic Insights

🕌 A complete Islamic website — Quran 30 Para with Audio, Namaz Timings, Duas, Ziyarat, Nahj al-Balagha, Ahadith, Islamic History, Qibla Direction.

**Live:** https://islamicinsights-net.netlify.app

## Pages
| Page | Description |
|------|-------------|
| `index.html` | Homepage |
| `quran.html` | Quran 30 Para with continuous audio, Arabic+Urdu+English |
| `namaz.html` | Prayer timings (GPS-based) |
| `qibla.html` | Qibla direction with compass |
| `duas.html` | Dua Kumayl, Tawassul, Nudba, Faraj |
| `ziyarat.html` | Ziyarat Ashura |
| `nahj.html` | Nahj al-Balagha (Sermons, Letters, Sayings) |
| `hadith.html` | Ahadith — Bukhari, Muslim, Tirmizi |
| `history.html` | Islamic History |
| `donate.html` | Donate (EasyPaisa/JazzCash/PayPal) |
| `about.html` | About page |

## Deploy to GitHub Pages
1. Upload all files to a GitHub repo
2. Go to Settings → Pages → Deploy from main branch
3. Your site is live!
